## Rust Pulsar Network

A connection Library for the Pulsar Network, a distributed hash table peer-to-peer messaging protocol written in Rust.

### Usage

In your `Cargo.toml`:

```

[dependencies]
pulsar-network = "0.1.2"

```

In your rust file:

```

use pulsar_network::{ Config, Network, Message, MessageKind };

```
